=================================================
|        COMMAND LINES TO LOAD APPLICATION      |
=================================================

1) create virtual environment

$ python -m venv tg_bot_venv

2) activate virtual environment

win: $ .\tg_bot_venv\Scripts\activate

lin/mac: $ source tg_bot_env/bin/activate

3) install packages

$ pip install -r requirements.txt

4) run bot.py file

$ python bot.py
